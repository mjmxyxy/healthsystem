Sprint 0: 项目准备（2 天）
- 确认需求、数据库设计、部署方案
- 初始化 Git 仓库、CI 配置（可选）

Sprint 1 (Week 1): 骨架与认证
- 后端：Spring Boot 项目搭建、MyBatis-Plus 集成、User 实体、注册/登录接口、BCrypt
- 前端：Vue3 项目搭建、登录/注册页、学号认证页面
- 环境：Docker Compose 配置完成，MySQL/Redis 可启动

Sprint 2 (Week 2): 测评与预约基础
- 后端：测评模块（SCL-90、SDS、SAS）数据模型、计分逻辑、报告生成 API
- 后端：预约模块（创建/冲突检测/分布式锁示例/定时任务）
- 前端：测评分页、预约流程、我的预约页

Sprint 3 (Week 3): 实时聊天与记录
- 后端：WebSocket 聊天模块、消息 AES 存储、导出 PDF 接口
- 前端：聊天页面、消息历史、导出功能

Sprint 4 (Week 4): 咨询师与管理员功能
- 后端：咨询师资料、资质审核、文章模块、管理员审核接口
- 前端：咨询师工作台、管理后台（简易）

上线准备（Week 5）
- 安全加固（HTTPS、密钥管理、日志审计）
- 性能测试、Bug 修复
- 部署文档与运维手册
